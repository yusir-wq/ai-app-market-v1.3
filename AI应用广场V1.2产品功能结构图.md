## 4. 业务流程

### 4.1 首页发送消息流程

```
用户进入首页
    │
    ▼
选择推荐模型（可选，toggle选中）
    │
    ▼
在输入框输入消息
    │
    ▼
点击 @ 按钮选择模型（1~4个，显示为Pills）
    │
    ▼
配置参数（可选：联网搜索/深度思考/图片参数/视频参数）
    │
    ▼
上传附件或参考图（可选）
    │
    ▼
点击发送 ⬆️
    │
    ├─ 未登录 → 弹出登录窗口
    └─ 已登录 → 校验必填参考图
              │
              ▼
        创建用户消息 + AI回复卡片
              │
              ▼
        跳转到对话页，显示多模型回复
```

### 4.2 多模型对话流程

```
用户发送消息 + @模型（1~4个）
    │
    ▼
创建用户消息气泡（右侧，显示@模型名）
    │
    ▼
为每个模型创建独立AI回复卡片
    │
    ├─ 加载态：显示"思考中…" + 骨架屏
    └─ 并行生成（staggered：800ms + index×1200ms）
    │
    ▼
全部完成后输入框恢复可用
    │
    ▼
用户可执行：复制 / 重新生成 / 回复（针对单个模型）
    │
    ▼
点击"回复" → 后续只针对此模型
```

### 4.3 联网搜索展示流程

```
用户启用联网搜索 → 发送消息
    │
    ▼
AI回复卡片生成
    │
    ▼
卡片底部显示 "引用来源 (N)"
    │
    ▼
用户点击 → 右侧抽屉滑出（380px）
    │
    ▼
显示搜索结果列表（网站名/标题/简介/URL）
    │
    ▼
点击URL → 新标签页打开
    │
    ▼
点击遮罩/关闭 → 抽屉关闭
```

### 4.4 深度思考展示流程

```
用户启用深度思考 → 发送消息
    │
    ▼
AI回复卡片生成
    │
    ▼
卡片正文前显示"深度思考过程"（默认折叠）
    │
    ▼
用户点击展开 ▼ → 显示推理过程文本
    │
    ▼
```

## 6. 状态设计

### 6.1 对话状态（Conversation）

```
[新建] → [进行中] → [已完成]
            │
            ↓
         [已归档]
            │
            ↓
         [已删除]（软删除）
```

| 状态 | 说明 |
|------|------|
| 新建 | 对话刚创建，无消息 |
| 进行中 | 有消息交互 |
| 已完成 | 对话结束 |
| 已归档 | 用户手动归档 |
| 已删除 | 用户删除，软删除 |

### 6.2 消息状态（Message）

```
[发送中] → [成功]
    │
    ↓
[失败] → [重试]
```

| 状态 | 说明 |
|------|------|
| 发送中 | 消息正在发送/生成中 |
| 成功 | 消息发送/生成成功 |
| 失败 | 发送/生成失败，可重试 |

### 6.3 AI 回复状态（Assistant Message）

```
[思考中] → [生成中] → [成功]
    │           │
    ↓           ↓
[失败]      [失败]
```

| 状态 | 说明 |
|------|------|
| 思考中 | 模型正在思考（骨架屏+"思考中…"） |
| 生成中 | 模型正在生成回复内容 |
| 成功 | 回复生成完成 |
| 失败 | 生成失败，显示错误信息 |

### 6.4 邀请奖励状态（InviteRecord）

```
[待发放] → [已发放]
```

| 状态 | 说明 |
|------|------|
| 待发放 | 被邀请人已注册，奖励待发放 |
| 已发放 | 奖励已发放到双方账户 |

### 6.5 MCP 服务状态（MCPService）

```
[已启用] ↔ [已停用]
```

| 状态 | 说明 |
|------|------|
| 已启用 | MCP服务可用，可被勾选 |
| 已停用 | MCP服务不可用，不可勾选 |

### 6.6 消费记录状态（UsageRecord）

```
[生成中] → [成功]
    │
    ↓
[失败]
```

| 状态 | 说明 |
|------|------|
| 生成中 | 内容正在生成 |
| 成功 | 生成成功，已扣费 |
| 失败 | 生成失败，未扣费/退费 |

### 6.7 左侧导航状态（NavPanel）

```
[展开态 288px] ↔ [折叠态 56px]
```

| 状态 | 说明 |
|------|------|
| 展开态 | 显示完整内容：品牌区、按钮、对话列表、功能入口、用户信息 |
| 折叠态 | 仅显示图标列 + Tooltip |
